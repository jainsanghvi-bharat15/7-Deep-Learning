# Deep Learning Projects using Artificial Neural Networks (ANN)

This repository contains beginner-friendly Deep Learning projects built using TensorFlow, Keras, Scikit-Learn, NumPy, Pandas, Matplotlib, and Seaborn. The projects demonstrate ANN applications in regression, classification, image recognition, healthcare, education, environmental monitoring, and price prediction.

## Projects

### 1. Air Pollution Level Estimation (PM2.5) using ANN

Predicts PM2.5 concentration using weather and environmental parameters such as temperature, pressure, humidity, rainfall, and wind speed.

**Techniques Used:** Regression, ANN, Environmental Analytics

---

### 2. Heart Disease Prediction using ANN

Predicts the likelihood of heart disease using medical parameters such as age, cholesterol, blood pressure, heart rate, and ECG results.

**Techniques Used:** Binary Classification, ANN, Confusion Matrix

---

### 3. Student Placement Prediction using ANN

Predicts whether a student will be placed based on academic performance, test scores, and employability-related attributes.

**Techniques Used:** Binary Classification, ANN, Data Preprocessing

---

### 4. Used Car Price Prediction using ANN

Estimates the selling price of used cars using features such as vehicle age, mileage, fuel type, transmission, and ownership history.

**Techniques Used:** Regression, ANN, Feature Encoding

---

### 5. Electricity Bill Prediction using ANN

Predicts monthly electricity bills based on units consumed, household size, AC usage, geyser usage, and solar panel availability.

**Techniques Used:** ANN, Regression, Feature Scaling

---

### 6. Handwritten Digit Recognition using ANN

Classifies handwritten digits (0–9) from the MNIST dataset using a feed-forward neural network.

**Techniques Used:** Multi-Class Classification, ANN, Computer Vision

---

### 7. Fashion Image Classification using ANN

Classifies clothing images from the Fashion-MNIST dataset into categories such as shirts, shoes, dresses, bags, and sneakers.

**Techniques Used:** Image Classification, ANN, Deep Learning

---
